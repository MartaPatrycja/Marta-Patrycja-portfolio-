Weather app

Current weather application is second project that I have been working on during SheCodes classes. It was build using weather geolocation API: openweathermap.org.
You can choose any city in the world to check daily and hourly celsius degrees, wind, humidity and pressure. As a bonus you have a 5 forthcomming days forecast. The application displays current date and time. You can also change city or switch to current location.

Superdog app

Nola Superdog is my first website. Navigation is simple and clean, built with HTML and CSS. It is about dog called Nola, who can be taken home from a shelter. It can work as a pattern for more complex websites.
While using this page you can find out wheter you can adopt this sweet little puppy. Just press "take me home" button.
Website has been coded while learning at SheCodes Pro school, where Matt Delac teaches women front-end development.
