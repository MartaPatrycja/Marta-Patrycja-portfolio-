JavaWeather

The weather app is my second project at SheCodes. It was built using tools such as: HTML, CSS, JavaScript and the WeatherGeolocation API: openweathermap.org.

The application displays the current date, time and weather of any city in the world. Tells you about wind, humidity and pressure. As a bonus, you can see the upcoming hourly and daily forecast.
